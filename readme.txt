== Description ==

The WP Mass Delete plugin allows you to mass delete WordPress posts and pages according to the specified conditions such as date range, post status, post content, etc.

Please use it carefully!

More usage info can be found at the [plugin's homepage](http://www.cyberseo.net/wp-mass-delete/ "WP Mass Delete").

== Installation ==

1. Upload 'wp-flash.php' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
